# 🚀 TaskFlow API: Система управления проектами и задачами
#### Pet-project
**TaskFlow** — это бэкенд-сервис для управления проектами, задачами и подзадачами, построенный на современном стеке Python. Это продуманная иерархическая система, позволяющая структурировать работу по проектам и отслеживать прогресс на всех уровнях.

## 🎯 Ключевые возможности

### Для пользователя
- **🏗️ Иерархия "Проект → Задача → Подзадача"**: Полноценная структура для управления сложными рабочими процессами
- **👤 Многопользовательская среда**: Каждый пользователь видит и управляет только своими данными
- **📊 Управление статусами и приоритетами**: Гибкая система для отслеживания прогресса (new, in_progress, done) и важности (low, middle, high)
- **🛡️ Безопасная аутентификация**: JWT-токены для защиты всех эндпоинтов
- **🗑️ Безопасное удаление**: Подтверждение перед удалением критичных сущностей (`confirm = true | false`)

### Для разработчика
- **🚀 Готовая к запуску архитектура**: Чистое разделение на слои API, бизнес-логики и данных
- **📚 Автоматическая документация**: Swagger UI и ReDoc доступны сразу после запуска
- **🐳 Docker-готовность**: Простой деплой через Docker
- **🗄️ Миграции БД**: Управление схемой через Alembic
- **🔌 Гибкость**: Легкая замена SQLite на PostgreSQL

## 🛠️ Технологический стек (Tech Stack)

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| **Язык** | Python 3.11+ | Основной язык разработки |
| **Веб-фреймворк** | FastAPI 0.115.0 | Высокопроизводительный фреймворк для API |
| **ORM** | SQLAlchemy 2.0.35 | Работа с базой данных через объектную модель |
| **Миграции** | Alembic | Управление изменениями схемы БД |
| **Аутентификация** | AuthX (JWT) | Безопасная аутентификация через токены |
| **Валидация** | Pydantic 2.9.0 | Валидация и сериализация данных |
| **База данных** | SQLite | БД для разработки |
| **Сервер** | Uvicorn | ASGI-сервер для запуска приложения |
| **Контейнеризация** | Docker | Упрощение деплоя и изоляция окружения |

## ⚙️ Установка и запуск

### Локальный запуск (для разработки)

```bash
git clone https://github.com/PierPankratc/Todo-list.git
cd Todo-list

python -m venv .venv
source .venv/bin/activate  # Для Mac/Linux
# или
.venv\Scripts\activate      # Для Windows

pip install -r requirements.txt

uvicorn main:app --reload

    Swagger UI: http://localhost:8000/docs

    ReDoc: http://localhost:8000/redoc

## Через Docker
```
docker build -t taskflow-api .

docker run -d -p 8000:8000 --name taskflow taskflow-api

docker stop taskflow
docker rm taskflow
```
